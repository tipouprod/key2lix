# اختبارات E2E — Playwright (P27)

اختبارات نهاية إلى نهاية باستخدام Playwright للتأكد من عمل الصفحات الرئيسية وواجهة API.

---

## 1. التشغيل

```bash
# تثبيت المتصفحات (مرة واحدة)
npx playwright install chromium

# تشغيل الاختبارات
npm run test:e2e
```

---

## 2. الملفات

| الملف | الدور |
|-------|-------|
| `playwright.config.js` | إعدادات Playwright، تشغيل الخادم تلقائياً |
| `e2e/smoke.spec.js` | اختبارات أساسية: الصفحات، `/health` |

---

## 3. الاختبارات الحالية

- تحميل الصفحة الرئيسية
- تحميل صفحة المنتجات
- تحميل صفحة السلة
- تحميل صفحة الدعم
- استجابة نقطة `/health` (status, db)

---

## 4. المتغيرات

- **BASE_URL:** الرابط الأساسي (افتراضي: `http://localhost:3000`)
- **CI:** عند ضبطه ينتظر الخادم ولا يعيد استخدام خادم موجود
